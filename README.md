# 💰 Finanças Bento Grid

Aplicativo de gestão financeira com interface acolhedora e design Bento Grid.

## 🚀 Acesse o App

[https://seu-usuario.github.io/meu-app-financas](https://seu-usuario.github.io/meu-app-financas)

## 📸 Demonstração

![Dashboard do App](https://via.placeholder.com/800x400.png?text=Dashboard+Financeiro)

## ✨ Funcionalidades

- ✅ Configuração inicial de categorias de gastos
- ✅ Definição de fontes de renda
- ✅ Registro de transações (receitas/despesas)
- ✅ Acompanhamento de recebimentos pendentes
- ✅ Agendamento de recebimentos futuros
- ✅ Alertas de orçamento excedido
- ✅ Dashboard com visualização em Bento Grid
- ✅ Gráfico de evolução financeira
- ✅ Persistência de dados via LocalStorage

## 🛠️ Tecnologias Utilizadas

- **React 18** - Biblioteca para construção da interface
- **Tailwind CSS** - Framework de estilização
- **Babel Standalone** - Compilação JSX em tempo real
- **LocalStorage** - Persistência de dados
- **GitHub Pages** - Hospedagem gratuita

## 🎨 Design

- Interface acolhedora com cores pastel
- Tipografia Quicksand e JetBrains Mono
- Animações suaves
- Layout responsivo Bento Grid
